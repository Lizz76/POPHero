# POPHero Roguelike Marble Prototype

这是一个用于验证核心乐趣的最小可玩原型：

- 玩家从底部拖拽瞄准并发射单颗小球
- 发射前显示绿色预测轨迹
- 小球在上/左/右墙与方块之间反弹
- 命中攻击 / 倍率 / 护盾方块后累计本轮收益
- 小球落到底部后结束回合并结算对上方敌人的伤害
- 敌人死亡后发放金币、回血，并进入 Buff 三选一
- 下一轮会根据上次落点继续发射，场内方块随机重排

## 当前实现

- `PopHeroGame`
  - 运行时自建场景、状态机、敌人推进、回合切换
- `BallController`
  - 单球飞行、命中判定、到底结算
- `PlayerLauncher`
  - 鼠标/触屏拖拽瞄准与发射
- `TrajectoryPredictor`
  - 发射前反弹轨迹预测
- `BoardManager`
  - 方块蓝图、随机重排、出口保护区
- `BoardBlock`
  - 攻击加法 / 攻击倍率 / 护盾三类方块
- `RoundController`
  - 本轮攻击 / 护盾 / 命中次数统计与结算
- `BuffManager`
  - 击杀后 Buff 三选一与永久强化应用
- `EnemyController`
  - 上层敌人展示与受击反馈
- `PopHeroHud`
  - 信息面板、Buff 面板与调试入口

## 默认参数

- 玩家：100 HP / 0 Shield / 0 Gold
- 小球：半径 `0.15`，速度 `12`
- 方块：
  - 攻击加法 `x4`
  - 攻击倍率 `x2`
  - 护盾 `x3`
- 敌人序列：
  - 50 / 80 / 120 / 170 / 250 HP

## 如何试玩

1. 打开 `SampleScene`
2. 进入 Play
3. 按住底部小球拖拽瞄准
4. 松手发射，观察轨迹与碰撞收益
5. 等球落到底部，查看回合结算
6. 击杀敌人后选择一个 Buff，继续下一轮

## 调试按钮

- 一键重排
- `+25 Gold`
- 秒杀敌人
- 玩家 `-10 HP`

## 目标

用最小成本验证这几个问题是否成立：

- 弹道预判 + 回合结算是否有乐趣
- 攻击与护盾双资源是否能形成决策
- 方块重排是否能带来“重新解题”的新鲜感
- Buff 作用于方块是否足够有成长感
